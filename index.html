Can you spot the phish?

Can you tell which domains are from example.com, and which are not?
This is a self-paced test to see if you can find the spoofed domains.

How to play

Look over each domain one-by-one.
If the is example.com, www.example.com, or secure.example.com, it is legitimate.
Once you have decided if it is phish check the checkbox.
If it turns green, it was legit.
If it turns red, it was phish.
Don't feel bad if you miss one; many of these have extremely small or hard to spot differences.
